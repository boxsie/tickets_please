## Single-user mode

Per discussion (see comment on [[github-google-oauth-providers]]): when no OAuth providers are configured, this middleware short-circuits to a synthetic "local" user with implicit owner-of-everything membership. Tests should cover that path explicitly so we don't accidentally break the dogfood/stdio experience.
