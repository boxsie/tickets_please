## Learnings
PromoteIdea (new svc/ideation.go) is flipArchive cloned: re-read ticket.yaml off disk → flip → stage yaml+comment in one StageOp → Commit under LockProject flock → enqueue comment embed → apply to cache → publish. Two key choices: (1) persist rec.Kind = KindWork.Stored() = "" so a promoted ticket is byte-identical on disk to a native work ticket (no lingering `kind: work` key). (2) Optional phase: rather than replicate the phase dir-move, PromoteIdea delegates to the existing AssignTicketToPhase AFTER the flip — but first compares the resolved target phase id to the promoted ticket's current PhaseID and skips if equal, to dodge AssignTicketToPhase's no-op-reject (ErrInvalidArgument). That means a phase-promote is TWO commits (promote + assign) — fine for hobby scale. New comment kind CommentKindSystemPromote + eventbus KindTicketPromoted (for T6 web live-patch).
