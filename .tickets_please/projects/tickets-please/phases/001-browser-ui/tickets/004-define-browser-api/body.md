Design and implement the local JSON API used by the browser server. Cover projects, phases, waves, tickets, comments, search, move, and complete. Handlers should call svc.Service directly, map domain errors to clear HTTP status and JSON payloads, and preserve the same server-side validation used by MCP tools. Acceptance criteria: a documented endpoint list exists; read endpoints return stable formatted JSON; mutating endpoints attach the web agent session; move and complete endpoints enforce existing comment and completion rules; handler tests cover success paths and representative domain errors.
